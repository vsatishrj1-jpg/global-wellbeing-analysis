# Global Well-Being Analysis — Data Visualization

An end-to-end data engineering and visualization project that consolidates global well-being data from multiple sources, processes it on the Microsoft Azure cloud, and surfaces insights through interactive Power BI dashboards.

## Overview

This project builds a complete cloud data pipeline to analyse global well-being and behavioural patterns. Data from 10+ heterogeneous sources is ingested into Microsoft Azure, transformed in Azure Databricks, and modelled in Power BI to reveal regional trends in happiness and well-being.

## Architecture & Workflow

1. **Ingestion** — 10+ heterogeneous data sources consolidated into Microsoft Azure.
2. **Orchestration** — the workflow is automated end-to-end using **Azure Data Factory**.
3. **Transformation** — data is cleaned and refined in **Azure Databricks** using **Spark SQL**, supporting root-cause analysis of behavioural patterns.
4. **Modelling & Visualisation** — refined data is modelled in **Power BI** with **DAX** measures and calculated columns.
5. **Reporting** — interactive dashboards surface key insights for end users.

## Key Insights

- **85%** reported happiness across **Asia**
- **45%** improvement in well-being across **South Asia**, driven by economic growth

## Tech Stack

- **Cloud platform:** Microsoft Azure
- **Orchestration:** Azure Data Factory
- **Processing:** Azure Databricks (Spark SQL)
- **Visualisation:** Power BI (DAX measures, calculated columns)

## Features

- Automated, end-to-end pipeline from raw sources to dashboards
- Consolidation of 10+ heterogeneous data sources
- Spark SQL transformations for root-cause analysis of behavioural patterns
- Interactive Power BI dashboards with drill-down insights by region

## Outcome

The project delivers a scalable cloud pipeline and a set of interactive dashboards that make global well-being trends easy to explore, highlighting regional differences in happiness and the role of economic growth in improving well-being.
